# 一、例子
## 1.1、get_Started

编译代码：
```bash
cd get_Started
cmake -S . -B build
cmake --build build
```
运行：
```bash
./build/get_Started
```
结果：
```bash
Received Sensor Data: ID=1, Temp=36.5
Received Sensor Data: ID=2, Temp=22.3
Received System Error: Overheating detected (Code: 408)
```
